git的安装

发表于 2017-04-26 | 分类于 前端 |

本篇主要讲解git的安装配置

简介

Git是一款免费、开源的分布式版本控制系统，用于敏捷高效地处理任何或小或大的项目。
Git是一个开源的分布式版本控制系统，可以有效、高速的处理从很小到非常大的项目版本管理。
Git 是 Linus Torvalds 为了帮助管理 Linux 内核开发而开发的一个开放源码的版本控制软件。

下载与安装

首先你可以打开命令行输入 git version

我们可以通过查看版本号的方法来判断电脑是否安装git

如果你跟上边显示的结果一样说明你电脑已经安装成。如果不是，请跟随我一起安装。

下载

点击Git进入官网下载最新版。


选择你电脑的系统的git版本；
1.Git for Windows Setup ——git的window版本
2.Git for Windows Portable (“thumbdrive edition”) ——git的window便携式版本（U盘）

如何查看自己的电脑版本号

1 计算机右键单击，选择属性。在系统栏目里有系统类型；显示的就是你的系统位数。
2 命令行输入 systeminfo命令，等待查询电脑配置信息。在随后出来的信息中的系统类型就是你的系统位数。

这里我的电脑是window64位；所以选择64位的git版本下载；等待下载完成。

一直点击NEXT直到出现选择。
在Windows Explorer integration选项中将“Git Bash here”和“Git GUI here”打对勾。



在“Adjusting your PATH environment”选项中，默认即可。



在“Configuring the line ending conversions”选项中，
第一个选项：如果是跨平台项目，在windows系统安装，选择；
第二个选项：如果是跨平台项目，在Unix系统安装，选择；
第三个选项：非跨平台项目，选择。



选择“Finish”，结束安装。


测试

这里我们的git已经成功安装；接下来我们测试一下；

首先你可以打开命令行输入 git version


我们可以通过查看版本号的方法来判断电脑是否安装git

如果你跟上边显示的结果一样说明已经成功安装。如果出错，请仔细检查步骤。重新再来。
