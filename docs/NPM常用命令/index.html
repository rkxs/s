NPM常用命令

发表于 2018-06-07 | 分类于 npm |

本篇讲述了npm命令大全

介绍

NPM是随同NodeJS一起安装的包管理工具，能解决NodeJS代码部署上的很多问题

作用

允许用户从NPM服务器下载别人编写的第三方包到本地使用。
允许用户从NPM服务器下载并安装别人编写的命令行程序到本地使用。
允许用户将自己编写的包或命令行程序上传到NPM服务器供别人使用。

使用

请确保你电脑已经装了node,npm

可使用

来检测

init

新建一个package.json文件 会有一些问题，根据你的答案会生成不同的json文件

tip

我们为了方便可使用

快速创建一个package.json 当然都是默认你选了yes的情况下

install

remove或者是uninstall

移除某个依赖

list

查看当前目录下所有的依赖

info

查看某一个依赖的信息

包括 基本说明，版本号等

outdated

查看当前目录下的依赖是否需要更新

update

更新当前目录

更新

全局更新npm

可使用 npm -v 来查看版本检测是否更新成功
